# 🌍 Global Tourism Receipts Analysis (200-2020)

A comprehensive data analysis project examing international tourism receipts trends, growth patterns and COVID-19 impact across 200+ countries using Python, Pandas and Seaborn.

## 📊 Project Overview
This project analysis **tourism receipts data** from a [Kaggle dataset](https://www.kaggle.com/datasets/abdulhamitcelik/international-tourism-receipts), tracking tourism revenue from 1995 to 2020.  The analysis reveals global trends, identifies top-performing countries and quantifies the dramatic impact of COVID-19 on the tourism industry.

### Key Questions Answered
- Which countries earn the most from international tourism?
- How did COVID-19 affect global tourism receipts?
- Which countries showed the fastes pre-pandemic growth?
- What countries proved most resilent during the 2020 crisis?
- How volatile are tourism receipts across different nations?

### Dataset Summary
| Feature | Details|
|---------|--------|
| **Source** | Kaggle|
|**Records**| 4,156 rows|
|**Countries**| 200+|
|**Time Period** | 2008 2020|
|**Key Metrics*** | Tourism receipts in USD |

## 🚀 How to Run This Project

### Prerequisites

```bash
Python 3.8 or higher
pip package manager
```

### Installation
1. **Clone repository**

    ```bash
    git clone https://github.com/lizkodjo/TourismData.git
    cd TourismData
    ```
2. **Install required packages**

    ```bash
    pip install -r requirments.txt

    # Or install individually:
    pip install pandas numpy seaborn matplotlib jupyter
    ```

3. **Launch Jupyter Notebook**
    ```bash
    jupyter notebook
    ```

4. **Open and run** `TourismData.ipynb`
